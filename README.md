# Unreels - SNS Focus Browser

## 🚀 프로젝트 소개
**Unreels**는 인스타그램의 릴스(Reels) 무한 루프에서 당신을 해방시켜주는 디지털 웰빙 브라우저입니다. 숏폼 중독 요소를 차단하고, 본래의 목적인 **소통(DM)**에만 집중할 수 있는 환경을 제공합니다.

## ✨ 주요 기능
- **DM 전용 모드**: 인스타그램 접속 시 자동으로 DM 인박스로 이동하며, 홈/탐색 페이지 접근을 차단합니다.
- **릴스 스와이프 차단**: DM에서 받은 릴스 시청은 가능하지만, 스와이프하여 다른 릴스로 넘어가는 것을 물리적으로 차단합니다.
- **간편 로그아웃**: 로그인 모달 내에 커스텀 로그아웃 버튼을 주입하여 편리한 계정 관리를 지원합니다.
- **심플한 인터페이스**: 불필요한 새로고침 버튼 등을 제거하여 온전한 소통에만 집중하게 합니다.
